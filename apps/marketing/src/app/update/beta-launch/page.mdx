# VaultKey Is Now in Beta

Today we're opening VaultKey to everyone. You can sign up, create your organization, and start building with our wallet infrastructure right now.

## What VaultKey does

VaultKey lets you create and manage wallets for your users without dealing with key management, chain integrations, or signing infrastructure yourself. You get a simple API and typed SDKs — we handle the rest.

With VaultKey you can:

- **Create wallets** for your users across EVM chains, Solana, and Tron.
- **Sign messages and transactions** asynchronously — submit a job, poll for the result.
- **Transfer stablecoins** (USDC, USDT) with optional gasless support via our relayer.
- **Sweep funds** to a master wallet on demand.
- **Check balances** for native tokens and stablecoins across any supported chain.

Private keys are encrypted at rest using a KMS provider and never stored or logged in plaintext. Signing happens in memory and the decrypted key material is discarded immediately after.

## Supported networks

- **EVM** — Ethereum, Polygon, Arbitrum, Base, Optimism, Avalanche, BSC, Linea, Scroll, zkSync, and their testnets.
- **Solana** — Mainnet and Devnet.
- **Tron** — Mainnet.

## SDKs

We're launching with official SDKs for TypeScript and Python so you can get up and running without touching raw HTTP.

```ts
import { VaultKey } from "@vaultkey/sdk";

const vk = new VaultKey({ apiKey: "vk_live_...", apiSecret: "..." });

const { data: wallet } = await vk.wallets.create({
  userId: "user_123",
  chainType: "evm",
});
```

```python
from vaultkey import VaultKey

vk = VaultKey(api_key="vk_live_...", api_secret="...")
wallet, err = vk.wallets.create({"user_id": "user_123", "chain_type": "evm"})
```

We are starting with a free tier so you can explore the platform without a credit card. Paid credits unlock higher limits and additional operations.
